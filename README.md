# To-Do List Application

An interactive and user-friendly To-Do List application built using HTML, CSS, and JavaScript. This project helps users organize and manage daily tasks efficiently with features such as task categorization, filtering, dark mode, and local storage support.

## 📌 Project Overview

This project is designed to improve productivity by allowing users to create, manage, and track their tasks in a simple and organized environment.

Users can add tasks, assign categories, edit existing tasks, mark tasks as completed, filter tasks based on their status, and permanently store their data in the browser.

The application provides a clean interface that makes task management easy and accessible.

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Categorize tasks
* Filter tasks (All, Active, Completed)
* Clear all completed tasks
* Dark mode toggle
* Real-time task counter
* Automatic data saving using Local Storage
* Responsive and clean user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JavaScript)
* Browser Local Storage
