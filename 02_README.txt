# To-Do List App

A simple To-Do List application built with HTML, CSS, and JavaScript.  
You can add tasks, mark them as completed, and delete them.  
This project uses localStorage so your tasks are saved even after refreshing the page.

---

## Project Structure
.
├── 01_index.html   # Main HTML file
├── 01_styles.css   # Styling for the app
└── 01_app.js       # JavaScript logic

---

## How to Run
1. Clone or download this repository.
2. Open the project folder in VS Code.
3. Open `01_index.html` in your browser (just double-click or use Live Server in VS Code).
4. Start adding your tasks!

---

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Saves tasks using localStorage
